# PostCSS config

My PostCSS config.

## Usage

Install:

```sh
npm install --save-dev @a01sa01to/postcss-config
```

Edit `postcss.config.mts`:

```js
import postcssConfig from '@a01sa01to/postcss-config'

export default {
  ...postcssConfig,
  // overrides
}
```
