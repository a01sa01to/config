# prettier config

My prettier config.

## Usage

Install:

```sh
npm install --save-dev @a01sa01to/prettier-config
```

Edit `.prettierrc.mjs`:

```js
import prettierConfig from '@a01sa01to/prettier-config'

export default {
  ...prettierConfig,
  // overrides
}
```
